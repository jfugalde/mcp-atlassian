# Farmacias Macross - Client Project

This repository contains client-specific deliverables for Farmacias Macross.

## Structure
- `docs/pm/` – Project management documentation for Macross
- `docs/shopify-analysis/` – Shopify audit outputs and performance data (generated by RYU platform tools)
- `scripts/` – Jira import helpers and Macross automation scripts
- `admin-app/` – Shopify admin app (Macross)

## Analysis Tooling

**Shopify analysis tooling** is maintained in the [RYU Platform](https://github.com/jfugalde/ryu-platform) repository under `tools/shopify-analysis/`.

To run analysis for Macross:
1. Use the tooling from `ryu-platform/tools/shopify-analysis/`
2. Configure `SHOPIFY_SHOP_DOMAIN="macross-pharma.myshopify.com"` in your `.env`
3. Save outputs to this repository's `docs/shopify-analysis/` directory

See [RYU Platform - Shopify Analysis](https://github.com/jfugalde/ryu-platform/tree/main/tools/shopify-analysis) for full documentation.
